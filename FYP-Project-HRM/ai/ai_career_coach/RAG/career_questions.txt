## AI/ML Engineering
Description: Highest growth career in Pakistan for 2026
Skills Needed: Python, TensorFlow, PyTorch, Scikit-learn, Data Analysis
Timeline: 8-12 months
Salary Range: Rs. 200,000 - 600,000
Companies: Afiniti, Arbisoft, Systems Limited
Roadmap:
- Learn Python fundamentals (Month 1-2)
- Study Math: Linear Algebra, Statistics (Month 2-3)
- Learn ML algorithms with Scikit-learn (Month 3-5)
- Deep Learning with TensorFlow/PyTorch (Month 5-8)
- Build 3 AI projects for portfolio (Month 8-11)
- Apply for junior AI roles (Month 12)

## Cybersecurity
Description: Security analyst, ethical hacking and penetration testing
Skills Needed: Networking, Linux, Python, Ethical Hacking, SIEM tools
Timeline: 8-12 months
Salary Range: Rs. 150,000 - 500,000
Companies: NADRA, HBL, Jazz
Roadmap:
- Learn networking fundamentals (Month 1-2)
- Linux and command line (Month 2-3)
- Ethical hacking basics (Month 3-5)
- Get CompTIA Security+ certified (Month 5-6)
- Practice on HackTheBox/TryHackMe (Month 6-9)
- Specialize in one area (Month 9-11)
- Apply for security roles (Month 12)
