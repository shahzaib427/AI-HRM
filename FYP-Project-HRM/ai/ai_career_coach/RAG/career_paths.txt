# CAREER PATHS IN PAKISTAN

## Software Engineering
Description: Build and maintain software applications. High demand in Pakistan.
Skills Needed: JavaScript, Python, React, Git, DSA
Timeline: 1-2 years
Salary Range: Rs. 100,000 - 350,000
Companies: Systems Limited, Techlogix, Arbisoft
Roadmap:
- Month 1-2: Learn HTML, CSS, JavaScript basics
- Month 3-4: Master React.js and build projects
- Month 5-6: Learn Node.js and database
- Month 7-8: Build full-stack portfolio projects
- Month 9-12: Apply for jobs and prepare for interviews

## Data Science
Description: Analyze data and build ML models. Growing field in Pakistan.
Skills Needed: Python, SQL, Statistics, Pandas, Machine Learning
Timeline: 1-2 years
Salary Range: Rs. 150,000 - 400,000
Companies: Afiniti, Motive, 10Pearls
Roadmap:
- Month 1-2: Python programming fundamentals
- Month 3-4: Pandas, NumPy, Data Visualization
- Month 5-6: Machine Learning basics (Scikit-learn)
- Month 7-8: Deep Learning (TensorFlow/PyTorch)
- Month 9-12: Build ML portfolio and Kaggle projects
